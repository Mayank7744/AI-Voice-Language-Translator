# Real-Time Language Translator

This project implements a **Real-Time Language Translator** that converts spoken language into multiple languages instantly.

The system uses **speech recognition, machine translation, and text-to-speech synthesis** to translate audio input and generate translated speech output.

---

## Features

- Speech-to-text transcription
- Real-time language translation
- Text-to-speech audio output
- Multi-language support
- Interactive interface using Gradio

---

## Technologies Used

- Python
- AssemblyAI (Speech Recognition)
- Translate Library (Machine Translation)
- ElevenLabs (Text-to-Speech)
- Gradio (User Interface)

---

## System Workflow

Audio Input  
↓  
Speech Recognition  
↓  
Text Translation  
↓  
Text-to-Speech Conversion  
↓  
Translated Audio Output
